# 注：该文法不支持全局变量定义，也不支持函数嵌套定义，变量只能在函数内部定义，程序由一个或多个地位相同的函数组成
# 程序主体部分
程序 -> 函数定义
函数定义 -> 函数定义 函数定义
函数定义 -> 变量类型 id ( ) { 函数块 }
函数定义 -> 变量类型 id ( 传入参数 ) { 函数块 }

# 参数列表部分
传入参数 -> 变量类型 id
传入参数 -> 变量类型 id , 传入参数

# 函数块部分（声明、赋值、循环、分支、返回）
函数块 -> 函数块 函数块
函数块 -> 变量类型 id ;
函数块 -> id = 算术表达式 ;
函数块 -> while ( 布尔表达式 ) { 函数块 }
函数块 -> if ( 布尔表达式 ) { 函数块 }
函数块 -> if ( 布尔表达式 ) { 函数块 } else { 函数块 }
函数块 -> return ;
函数块 -> return 算术表达式 ;

# 声明（暂时不考虑指针和数组）
变量类型 -> int
变量类型 -> float

# 表达式相关
算术表达式 -> 算术表达式 算术运算符 算术表达式
算术表达式 -> - 算术表达式
算术表达式 -> ( 算术表达式 )
算术表达式 -> id
算术表达式 -> digit

布尔表达式 -> 算术表达式 比较运算符 算术表达式
布尔表达式 -> 布尔表达式 && 布尔表达式
布尔表达式 -> 布尔表达式 || 布尔表达式
布尔表达式 -> ! 布尔表达式
布尔表达式 -> ( 布尔表达式 )
布尔表达式 -> true
布尔表达式 -> false

# 运算符相关
比较运算符 -> <
比较运算符 -> >
比较运算符 -> <=
比较运算符 -> >=
比较运算符 -> ==
比较运算符 -> !=

算术运算符 -> +
算术运算符 -> -
算术运算符 -> *
算术运算符 -> /
